# вҡҷпёҸ Installation

Clone the repository

```bash
git clone https://github.com/The-Rajput-PS/CareerLaunch-AI.git
```

Go inside

```bash
cd CareerLaunch-AI
```

Install dependencies

```bash
npm install
```

Run Backend

```bash
cd server

npm install

node server.js
```

Run Frontend

Open

client/index.html

using Live Server.
